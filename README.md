# mini-fabric-whiteboard

使用fabric实现画板功能，支持画笔，绘制直线，矩形，圆形，文字，移动缩放画布，撤销重做，插入图片，清屏与保存功能

# 访问链接

https://enson0131.github.io/white-board/

# 启动

```shell
yarn
yarn dev
```

# 坑点
1. fabric 默认没有支持擦出功能，可以在 npm 包里面的 mixin 获取对应的擦出功能包
